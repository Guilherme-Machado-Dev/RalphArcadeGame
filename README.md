## 📖 Regras e Instruções
- ⏱️ **Tempo**: Você tem 60 segundos para acumular a maior pontuação possível.  
- 🎯 **Objetivo**: Clique no quadrado onde o inimigo aparece para ganhar pontos.  
- 💥 **Pontuação**: Cada acerto vale +1 ponto.  
- ❌ **Erros**: Se clicar no quadrado errado, você perde uma vida.  
- ❤️ **Vidas**: O jogador começa com 3 vidas. Ao perder todas, o jogo termina.  
- 🔊 **Som**: Cada acerto toca um efeito sonoro para confirmar a ação.  
- 🏆 **Fim de jogo**: O jogo acaba quando o tempo chega a zero ou quando todas as vidas são perdidas.  
- 🔄 **Reinício**: Após o fim, o jogo reinicia automaticamente para uma nova partida.

## 🔧 Possíveis Melhorias
1. 🎶 **Música de fundo**  
   Adicionar uma trilha sonora contínua para tornar a experiência mais imersiva.

2. ❌ **Efeito sonoro ao errar**  
   Incluir um som específico quando o jogador clicar no quadrado errado, reforçando o feedback.

3. ✨ **Efeito visual ao errar**  
   Fazer o quadrado **piscar em vermelho** (mudança temporária de cor) quando o jogador clicar no quadrado errado, destacando o erro de forma clara.

💡 Além dessas ideias, você pode **clonar ou fazer um fork** do projeto e usar a sua criatividade para implementar novas funcionalidades e melhorias!
